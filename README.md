# Vityarthi Library Management System

## Overview
A Java-based console application designed to manage library operations such as book tracking, member management, and issuing/returning books. This project demonstrates Object-Oriented Programming principles and file handling in Java.

## Features
- **CRUD Operations:** Create and search for books and members.
- **Transaction Handling:** Issue and return books with validation logic.
- **Data Persistence:** Uses `Serializable` to save state between runs.
- **Input Validation:** Handles invalid user inputs gracefully.

## Technologies Used
- **Language:** Java (JDK 8+)
- **Concepts:** OOP, Collections Framework, Exception Handling, File I/O (Serialization)
- **IDE:** VS Code / IntelliJ / Eclipse
